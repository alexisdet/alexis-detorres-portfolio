## Update the live link

GitHub Pages is connected to the `main` branch, so the live link stays the same and updates after changes are pushed.

To publish future edits from PowerShell:

```powershell
.\publish.ps1 "Describe your update"
```

That command commits the current files and pushes them to GitHub. GitHub Pages usually refreshes within a minute or two.
